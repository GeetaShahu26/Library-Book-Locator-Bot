🤖 RFID-Triggered Smart Line FollowerAn autonomous robotics project featuring a dual-controller architecture. This bot remains in standby mode until a genuine RFID tag is scanned, which then triggers a precise 2-second line-following sequence.🌟 Project VisionThe goal of this project is to create a "Secure Task Bot." In a warehouse or hospital setting, you don't want a robot moving constantly. Our vision is a system where movement is only authorized via RFID authentication, ensuring that the robot only operates when a specific user or "job card" is presented.⚠️ The Problem & SolutionThe Problem:During development, we encountered two major issues:Electronic Noise: The ESP8266 and Arduino shared a power rail, causing "ghost triggers" where the motors would start without an RFID scan due to static/voltage spikes.Signal Hanging: The RFID reader would occasionally hang or "double-scan," causing the bot to jitter or loop indefinitely.The Solution:Double-Check Logic: The Arduino code now includes a Noise Filter. It waits 50ms and checks the trigger pin a second time to ensure the signal is a real 300ms pulse, not a spike.Active Pull-Down: The ESP8266 code forces the trigger pin LOW at the start of every loop to drain any residual capacitance.State Management: We implemented a PICC_HaltA() command so the bot won't re-trigger if the card is left on the reader.🛠️ Hardware SetupComponent ListMicrocontrollers: ESP8266 (NodeMCU) & Arduino UnoSensors: MFRC522 RFID Module & 2x IR Line SensorsMotor Driver: L298N Dual H-BridgePower: 7.4V Li-ion battery with a 5V Step-down regulatorCircuit Diagram🔌 Pin MappingESP8266 PinComponentArduino PinComponentD8 (GPIO15)RFID SDAPin 5Trigger Input (from ESP)D5 (GPIO14)RFID SCKPin 2Left IR SensorD7 (GPIO13)RFID MOSIPin 3Right IR SensorD6 (GPIO12)RFID MISOPin 11, 10, 9Motor A (EN, IN1, IN2)D3 (GPIO0)RFID RSTPin 6, 8, 7Motor B (EN, IN3, IN4)D1 (GPIO5)Trigger OutGNDCommon Ground (Shared)🚀 How to UseWiring: Connect the components as shown in the schematic. Important: Connect the GND of the ESP8266 to the GND of the Arduino.Flash ESP8266: Upload the RFID controller code. It will handle the "Security Gate."Flash Arduino: Upload the Line Follower code.
